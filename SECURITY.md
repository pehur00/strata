# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 0.2.x   | Yes       |
| < 0.2   | No        |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Use [GitHub's private vulnerability reporting](https://github.com/pehur00/strata/security/advisories/new) to submit a report. You can expect:

- Acknowledgement within **48 hours**.
- A status update within **7 days** indicating whether the report is accepted or declined.
- A patch or workaround within **30 days** for confirmed high-severity issues.

Please include:
- A description of the vulnerability and its potential impact.
- Steps to reproduce or a proof-of-concept.
- Any suggested mitigations (optional but appreciated).

## Disclosure policy

We follow coordinated disclosure. We will credit reporters in the release notes unless you prefer to remain anonymous.
